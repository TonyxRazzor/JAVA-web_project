
# Проект BUYSELL

Проект BUYSELL - это веб-приложение для онлайн-торговли, разработанное на Java с использованием Spring Framework.

## Описание

BUYSELL предоставляет возможность пользователям создавать аккаунты, размещать объявления о продаже товаров.
А также просматривать и покупать товары других пользователей.

## Технологии

Проект реализован с использованием следующих технологий:

- Java
- Spring Boot
- Spring Security
- Spring Data JPA
- Freemarker
- MySQL
- HTML/CSS
- Bootstrap

## Установка и запуск

1. Клонируйте репозиторий на свой компьютер:

```
https://github.com/TonyxRazzor/JAVA-web_project.git
```

2. Убедитесь, что у вас установлены Java, Maven и MySQL.

3. Создайте базу данных MySQL с именем `buysell`.

4. Укажите настройки базы данных в файле `application.properties`.

5. Запустите приложение с помощью файла:

```
BuysellApplication.java
```

6. Откройте веб-браузер и перейдите по адресу:

```
http://localhost:8090
```
